## 插件
本项目使用到的插件  
shared_preferences: `^2.0.6` 保存城市id，主题，语言数据的持久化  
dio: `^4.0.0` 网络请求  
intl: `0.17.0` 日期格式化  
fluttertoast: `^8.0.7` toast通知    
get: `4.6.1` 状态管理
